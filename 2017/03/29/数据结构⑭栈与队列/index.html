数据结构⑭栈与队列

发表于 2017-03-29 14:34:48 | 分类于 数据结构 |

优化的顺序队列和循环队列确实提高了队列的效率，可是实现过程比较复杂，那么有没有一种即提高队列的效率实现过程也比较简单的方法来实现队列呢？本节就介绍使用栈的形式来实现队列。

栈与队列

实现思路

  • 创建两个栈用于实现队列：inStack和outStack
  • 当新元素入队时，将其压入instack中
  • 当元素需要出队时，有两种情况
    • 当outStack为空时，首先将inStack中的元素逐一弹出并压入outStack中，最后将outStack的栈顶元素弹出
    • 当outStack不为空时，直接将outStack的栈顶元素弹出
